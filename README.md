# TeamOD_Capstone (ATENDO)

**Atendo** is an **automated attendance-checking system** designed to replace the slow, manual process of using pen-and-paper sheets or third-party online forms. Using **custom-generated QR codes**, Atendo allows users to record attendance quickly, accurately, and conveniently by scanning these QR codes via camera. These QR IDs can be used in any event that makes use of **Atendo**, whether for classes, meetings, organizational activities, or events.

**Key Features**
1.	**User Accounts** --> Enables users to securely save, load, and manage their attendance records.
2.	**Event Creation** --> Users can create events and configure custom attendance parameters.
3.	**Data Filtering** --> Provides tools to filter, search, and generate queries from attendance data.
4.	**CSV File Handling** --> All data is stored in CSV format, ensuring full compatibility with Excel and other data-processing tools.
5.	**Flexible & Customizable** --> Designed to be user-friendly and adaptable, allowing automation of various attendance-related workflows.
